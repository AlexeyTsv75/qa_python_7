Проект «qa_python_7»

В проекте протестировано API учебного сервиса "Яндекс.Самокат"
Протестированы ручки:
1. Создание курьера
2. Логин курьера
3. Создание заказа
4. Список заказов


**Запуск тестов**

- запуск всех тестов:
  pytest -v tests/
- запуск выбранного теста
  pytest -v tests/"имя_выбранного_теста"
